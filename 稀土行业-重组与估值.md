# 稀土行业 - 重组与估值

## 稀土种类
 - 轻稀土
	- 来源：北方白云鄂博矿
	- 作用：**消费电子、新能源汽车、风电、变频空调**...
	- 主要企业：北方稀土
 - 重稀土
	- 来源：南方离子型稀土
	- 作用：主要用于**军工、航空航天、高端芯片**等对性能稳定性要求极高的领域，例如：**火箭陀螺仪、激光制导、高精度电机**...
	- 主要企业：中国稀土，中稀有色

## 稀土企业重组
 - 中国稀土
	- 重组限期：2027年1月前，完成资产注入
	- 重组方式：
		- 注入 广东稀土（中稀有色）外，其他重稀土资产，以及部分轻稀土资源。
		- 收购 盛和资源，收纳其国内稀土部分。盛和资源海外部分，可能独立成立：中稀海外
 - 中稀有色：
	- 重组限期：2028年左右，完成资产注入
	- 重组方式：
		- 注入 广东省内稀土资源（新矿）
